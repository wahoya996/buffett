# ROE 杜邦分析图表生成器

基于 **AKShare** 自动获取 A 股财务数据，计算杜邦分解并生成可视化分析图表。

## 功能

1. **自动获取数据** — 输入 6 位股票代码，自动拉取近 5 年年报数据
2. **杜邦分解** — 计算 `ROE = 销售净利率 × 总资产周转率 × 权益乘数`
3. **四面板图表** — 同时展示以下 4 个视图：
   - ROE 趋势图
   - 杜邦三因子（净利率/周转率/权益乘数）对比
   - ROE 实际值 vs 分解计算值
   - 各因素对 ROE 变动的微分归因贡献
4. **自动保存** — 图表以 `股票代码_ROE分析.png` 输出到脚本所在目录

## 依赖

- `akshare` — A 股财务数据接口（首次运行自动安装）
- `matplotlib` — 图表绘制
- `pandas` — 数据处理

## 使用方法

```bash
python roe_analysis.py
```

按提示输入 6 位股票代码（如 `600887` 伊利股份），程序自动完成：

1. 获取近 5 年财务摘要 + 资产负债表数据
2. 计算各年杜邦分解指标
3. 输出数据表格
4. 生成 4 面板分析图并保存

## 注意事项

- 仅支持 A 股（沪深京交易所）6 位数字代码
- 总资产周转率从资产负债表独立计算，而非从 ROE 反推，因此实际 ROE 与分解值存在微小差异（加权平均效应），属正常现象
- 图表使用中文字体（SimHei / Microsoft YaHei），若系统缺失对应字体可自行修改 `plt.rcParams['font.sans-serif']`

# 股票财务分析图表工具

基于 [akshare](https://github.com/akfamily/akshare) 获取 A 股上市公司年报数据，一键生成 **营业收入 · 净利润 · 同比增速** 双轴组合图表。

## 功能

- 输入任意 A 股代码（如 `600887`、`000858.SZ`），自动识别交易所
- 获取最近 **10 年** 年报数据：
  - 营业收入（亿元）
  - 归母净利润（亿元）
  - 营收同比增速（%）
  - 净利润同比增速（%）
- **双 Y 轴** 融合图表：
  - **左轴** — 柱状图：营业收入（蓝）、归母净利润（橙）
  - **右轴** — 折线图：营收同比增速（红实线）、净利润同比增速（绿虚线）
- 自动保存为 300 DPI 高清 PNG

## 环境要求

- Python ≥ 3.8
- 操作系统：Windows / macOS / Linux

## 安装

```bash
pip install akshare pandas matplotlib
```

## 使用

```bash
python stock_finance.py
```

按提示输入股票代码即可。

**股票代码格式**（自动识别交易所）：

| 输入示例         | 交易所   | 说明                   |
|------------------|----------|------------------------|
| `600887`         | 沪市     | 6 开头自动补 SH        |
| `000858`         | 深市     | 0/3 开头自动补 SZ      |
| `000858.SZ`      | 深市     | 带后缀亦可             |
| `SH600887`       | 沪市     | 已带前缀则直接使用     |
| `832317`         | 北交所   | 8/4 开头自动补 BJ      |

## 输出

图表 PNG 保存在脚本所在目录，命名格式：

```
{股票代码}_财务分析_YYYYMMDD.png
```

例如：`600887_财务分析_20260605.png`

## 注意事项

- 数据来源：**东方财富**，通过 akshare 实时获取，依赖网络连接
- 仅提取 **年报**（报告类型 = "年报"），不含季报/中报
- 营业收入/净利润单位为 **亿元**，同比增速单位为 **百分比（%）**
- 首次运行 akshare 时会缓存部分数据，后续查询更快
- 若图表中文显示为方框，请确保系统已安装中文字体（Windows 通常自带 SimHei，Linux 可安装 `fonts-wqy-microhei`）
